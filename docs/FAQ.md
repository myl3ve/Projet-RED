
# FAQ — Problèmes courants

**`unknown escape` dans un ASCII ?**  
→ Tu as un `\` non échappé. Utilise des raw strings `` `...` `` ou mets `\`.

**Couleurs ne s’affichent pas ?**  
→ Mets `useColor = true` (ou essaie Windows Terminal).

**ASCII trop large (cassé sur plusieurs lignes) ?**  
→ Élargis la console (voir `INSTALL.md`) ou utilise un titre compact.

**Import not used ?**  
→ Supprime l’import inutilisé.

**`go: command not found` ?**  
→ Installe Go et redémarre le terminal.
